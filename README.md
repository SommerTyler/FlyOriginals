# FlyOriginals Landingpage

Landingpage fuer FlyOriginals mit:

- Lade-Szene mit Logo
- Sektionen: `Über uns`, `Projekte`, `Partner`, `Socials`
- YouTube- und Twitch-Einbindung
- Kontaktformular mit Supabase-Integration
- Footer mit `Impressum` und `Datenschutz`

## Lokal starten

```bash
npm install
npm run dev
```

## Supabase konfigurieren

1. Datei `.env.example` nach `.env` kopieren.
2. Werte eintragen:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. In Supabase eine Tabelle `contact_requests` anlegen:

```sql
create table if not exists public.contact_requests (
  id bigint generated by default as identity primary key,
  name text not null,
  email text not null,
  message text not null,
  created_at timestamptz not null default now()
);
```

## Deploy mit Render (jetzt)

- **Build Command:** `npm run build`
- **Publish Directory:** `dist`
- **Environment Variables:** dieselben `VITE_*` Werte aus `.env`

## Umzug auf Vercel (spaeter)

- Projekt mit GitHub-Repo in Vercel importieren
- Framework: `Vite` (wird automatisch erkannt)
- Build: `npm run build`
- Output: `dist`
- Environment Variables (`VITE_*`) in Vercel setzen
- Eigene Domain im Vercel Dashboard verbinden

## Hinweis zu Twitch

Im Twitch-Embed muss der `parent` Parameter auf deine echte Domain angepasst werden, z. B.:

`https://player.twitch.tv/?channel=dein_channel&parent=deinedomain.de`
